# Portfolio - Wallace Luis

This is the professional portfolio of **Wallace Luis**, designed to highlight his two facets of expertise: **Full Stack Developer** and **Data Analyst**. The project offers an interactive and immersive experience, using modern technologies and artificial intelligence.

## üöÄ Technologies Used

The project was built with a robust stack focused on performance and UX:

*   **[Next.js 14+](https://nextjs.org/)**: React framework for production, using App Router and Server Components.

*   **[Tailwind CSS](https://tailwindcss.com/)**: Utility-first CSS for fast and responsive designs.
*   **[Framer Motion](https://www.framer.com/motion/)**: Library for fluid and complex animations.
*   **[Google Gemini AI](https://deepmind.google/technologies/gemini/)**: Integration with Gemini API for the intelligent chatbot.
*   **[Lucide React](https://lucide.dev/)**: Lightweight and consistent icons.
*   **Web Speech API**: For real-time voice recognition and transcription.

*   **Data Analyst Mode**: Focused on data (Python, SQL, Power BI), with Green/Emerald visual theme and data grid background.
*   The animated toggle allows for a smooth transition between the two personas.

### 2. AI Chatbot (WhatsApp Style)

*   **Realistic Design**: Interface identical to WhatsApp Web/Mobile.
*   **Generative AI**: Answers questions about Wallace's career and skills using Google Gemini.
*   **Voice Command**: Allows the user to send messages via voice (Speech-to-Text).
*   **Animations**: Visual feedback for "typing", online status, and message sending.
